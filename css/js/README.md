# LALID S Portfolio

## Setup
1. Put `LALID_S_Resume.pdf` inside `assets/`.
2. Replace `your_form_id` in the contact form with your Formspree ID.
3. Update GitHub and LinkedIn links in `index.html`.
4. Open `index.html` locally or deploy to GitHub Pages.

## Folder structure
portfolio/
├─ index.html
├─ css/
│  └─ styles.css
├─ js/
│  └─ script.js
└─ assets/
   └─ LALID_S_Resume.pdf

## Deploy to GitHub Pages
1. Push the folder to a GitHub repository.
2. Go to Settings → Pages.
3. Choose branch `main` and root `/`.
4. Save and wait for the public URL.

## Notes
- The contact form uses Formspree for static hosting support.
- The design includes smooth scrolling and reduced-motion accessibility support.
